# 新能源充电需求预测与布局优化  
（统计建模大赛作品）

本项目围绕城市新能源充电设施“供需错配”问题，构建了融合**气候、政策与用户行为**
等多源数据的数据分析与时间序列建模框架，对新能源充电需求进行预测，并对充电设施
布局方案进行量化评估与优化。

该项目为统计建模大赛参赛作品，重点关注数据分析过程、模型构建思路与结果可解释性，
而非单一模型效果。
## 项目背景

随着新能源汽车保有量快速增长，城市充电基础设施在空间布局和时间供给上
逐渐暴露出供需错配问题。传统规划方法难以同时考虑气候因素、政策影响以及
用户行为变化。

本项目尝试从数据分析的角度，构建多源数据融合模型，对新能源充电需求进行
预测，并为充电设施布局优化提供量化依据。
## 数据来源

- 国家及地方统计年鉴（新能源汽车保有量、人口数据等）
- 充电桩运营相关数据（充电需求时间序列）
- 气象数据（温度、降水等）
- 政策相关数据（补贴政策发布时间、节假日因素）

> 注：部分数据经过脱敏或模拟处理，仅用于学术研究与方法验证。
## 方法与技术路线

1. **数据清洗与特征构建**
   - 对多源时序数据进行清洗、缺失值处理与异常值修正
   - 构建气候、政策与行为相关特征变量

2. **影响因素分析**
   - 通过相关性分析与因果分析方法，评估政策与气候因素对充电需求的影响

3. **需求预测模型**
   - 使用 Prophet 对趋势与季节性进行建模
   - 结合 LSTM 对残差部分进行非线性建模，提升预测精度

4. **布局优化分析**
   - 基于预测结果，采用加权 TOPSIS 与 K-Means 聚类方法，对不同布局方案进行综合评估
## 实验结果与成果

- 预测模型在测试集上的 RMSE 相较基线 ARIMA 模型降低约 **36.5%**
- 分析结果验证了政策补贴对充电需求存在显著的滞后影响
- 布局优化方案在覆盖率与利用效率等指标上表现更优

**项目成果：**
